# ConceptGuide 

### An AI-Powered Learning Companion for Understanding Complex Concepts

**ConceptGuide** transforms passive reading into active understanding. Instead of struggling through complex PDFs alone, ConceptGuide acts as an intelligent tutor. It actively diagnoses your learning gaps, visualizes concept relationships, and creates personalized learning paths tailored to your specific needs.

---

##  Key Features

*   **Smart PDF Processing**  
    Upload any educational PDF. The system cleans, organizes, and prepares the text for interaction.
*   **Intelligent Confusion Detection**  
    Highlight text you don't understand to trigger a 5-level AI diagnostic process (checking vocabulary, prerequisites, misconceptions, etc.).
*   **Interactive Concept Maps**  
    Visualizes dependencies and related topics using ReactFlow, showing you exactly what you need to learn first.
*   **Personalized Learning Paths**  
    Generates a custom repair path with explanations, analogies, and practice problems to bridge your knowledge gaps.
*   **Voice & Text Input**  
    Explain your confusion naturally using speech-to-text or typing.
*   **Progress Tracking**  
    Track your mastery, study streaks, and session history via a detailed analytics dashboard.

---

##  How It Works

1.  **Upload & Read:** Upload your PDF document and start reading.
2.  **Flag Confusion:** Highlight a section you find difficult and click "I'm Confused."
3.  **Diagnose:** Answer 5 targeted questions generated by Gemini AI to pinpoint *why* you are confused.
4.  **Visualize:** View a generated Mind Map showing prerequisite concepts you might be missing.
5.  **Learn:** Follow a step-by-step Learning Path to master the missing foundations and return to your reading with confidence.
6.  **Track:** Set goals, check your analytics, and review previous sessions in your history.

##  Getting Started

### Prerequisites

*   **Node.js** (v18+)
*   **Google Gemini API Key**
*   **Firebase Project Credentials**

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/sreeC-26/ConceptGuide_v2.git
    cd ConceptGuide
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env` file in the root directory and add your credentials:

    ```env
    GEMINI_API_KEY=your_google_ai_key

    # Firebase Config
    VITE_FIREBASE_API_KEY=your_key
    VITE_FIREBASE_AUTH_DOMAIN=your_domain
    VITE_FIREBASE_PROJECT_ID=your_id
    # ... add remaining firebase config
    ```

### Run the Application

Open two terminals to run the backend and frontend simultaneously:

**Terminal 1: Backend**
```bash
npm run server
```
**Terminal 2: Frontend**
```bash
npm run dev
```
Visit http://localhost:5173 to start learning!
