ConceptGuide
An AI-Powered Learning Companion for Understanding Complex Concepts

What is ConceptGuide?
ConceptGuide transforms passive reading into active understanding. Instead of struggling through complex PDFs alone, ConceptGuide acts as an intelligent tutor. It actively diagnoses your learning gaps, visualizes concept relationships, and creates personalized learning paths tailored to your specific needs.

Key Features:
Smart PDF Processing: Upload any educational PDF. The system cleans, organizes, and prepares the text for interaction.
Intelligent Confusion Detection: Highlight text you don't understand to trigger a 5-level AI diagnostic process (checking vocabulary, prerequisites, misconceptions, etc.).
Interactive Concept Maps: Visualizes dependencies and related topics using ReactFlow, showing you exactly what you need to learn first.
Personalized Learning Paths: Generates a custom repair path with explanations, analogies, and practice problems to bridge your knowledge gaps.
Voice & Text Input: Explain your confusion naturally using speech-to-text or typing.
Progress Tracking: Track your mastery, study streaks, and session history via a detailed analytics dashboard.

How It Works:
Upload & Read: Upload your PDF document and start reading.
Flag Confusion: Highlight a section you find difficult and click "I'm Confused."
Diagnose: Answer 5 targeted questions generated by Gemini AI to pinpoint why you are confused.
Visualize: View a generated Mind Map showing prerequisite concepts you might be missing.
Learn: Follow a step-by-step Learning Path to master the missing foundations and return to your reading with confidence.
You can set goals for yourself, as well as check your analytics 
You can review your previous sessions in history

Technology Stack
Frontend: React, Vite, Tailwind CSS, Zustand, ReactFlow
Backend: Node.js, Express
AI Engine: Google Gemini AI (@google/generative-ai)
Database & Auth: Firebase
Utilities: PDF.js (Parsing), Vitest (Testing)

Getting Started

Prerequisites
Node.js (v18+)
Google Gemini API Key
Firebase Project Credentials

Installation

Clone the repository
git clone https://github.com/your-username/ConceptGuide.git
cd ConceptGuide

Install dependencies
npm install

Configure Environment
Create a .env file in the root directory:

.env
GEMINI_API_KEY=your_google_ai_key

# Firebase Config
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_id
# ... add remaining firebase config

Run the application
Open two terminals:
# Terminal 1: Backend
npm run server
# Terminal 2: Frontend
npm run dev

Visit http://localhost:5173 to start learning.
